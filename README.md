## GitHub User List

### Getting Started

Before starting this application you must start the [back-end](https://github.com/felipescunha/backend-nestjs)

Run yarn to download the project dependencies:
```bash
yarn
```

Run yarn dev to start the application:
```bash
yarn dev
```
